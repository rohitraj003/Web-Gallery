
# Image-Gallery
Our web app is your gateway to a visual journey where each image tells a unique story.


## Features

- See your images in different view such as
- Carousel view
- Scroll view
- Collage view



## Tech Stack

- Reactjs for frontend.
- Bootstrap for styling.
